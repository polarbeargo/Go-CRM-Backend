Go-CRM-Backend

RESTful server endpoints getCustomer, addCustomer, getCustomers, updateCustomer, deleteCustomer for CRUD operations

Endpoints:

  • getCustomer
  • addCustomer
  • getCustomers
  • updateCustomer
  • deleteCustomer

Example:

  • http://localhost:3000/getCustomer?id=1
  • http://localhost:3000/addCustomer?name=John&email=john@gmail&phone=1234567890
  • http://localhost:3000/getCustomers
  • http://localhost:3000/updateCustomer?id=1&name=John&email=john@gmail&phone=1234567890
  • http://localhost:3000/deleteCustomer?id=1
HTTP Method URL Path Parameters Response
GET /customers/{id} id Customer
POST /customers name, email, phone Customer
GET /customers Customers
PUT /customers/{id} id, name, email, phone Customer
DELETE /deleteCustomer/{id} id Customer
