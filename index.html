Go-CRM-Backend

RESTful server endpoints getCustomer, addCustomer, getCustomers, updateCustomer, deleteCustomer for CRUD operations

Endpoints:

  • getCustomer
  • addCustomer
  • getCustomers
  • updateCustomer
  • deleteCustomer
HTTP Method URL Path Description
GET /customers/{id} Getting a single customer
POST /customers Creating a customer
GET /customers Getting all customers
PUT /customers/{id} Updating a customer
DELETE /customers/{id} Deleting a customer
